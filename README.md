## üõ† Local Setup

1. **Clone repository**
```bash
git clone https://github.com/saqinnoor/Portfolio.git
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```

4. **Build for production**
```bash
npm run build
```
